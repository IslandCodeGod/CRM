# CRM系统

一个功能完整、界面友好的客户关系管理系统，基于Python Flask开发。

## 功能特性

### 核心功能
- ✅ **用户认证系统** - 登录、注册、会话管理
- ✅ **客户管理** - 完整的CRUD操作，支持搜索和筛选
- ✅ **销售线索跟踪** - 线索状态管理、评分系统
- ✅ **销售机会管理** - 看板视图、销售流程管理
- ✅ **联系人管理** - 客户联系人信息管理
- ✅ **任务管理** - 待办事项、优先级管理
- ✅ **数据分析** - 可视化图表、关键指标展示

### 技术亮点
- 🎨 **现代化UI设计** - 深色/浅色主题切换
- 📱 **响应式布局** - 支持移动端访问
- ⚡ **流畅动画** - 页面过渡和交互动画
- 📊 **数据可视化** - Chart.js图表展示
- 🔍 **实时搜索** - 智能搜索和筛选
- 🎯 **看板管理** - 拖拽式销售机会管理

## 技术栈

### 后端
- Python 3.x
- Flask 3.0
- SQLAlchemy (ORM)
- Flask-Login (认证)
- SQLite (数据库)

### 前端
- HTML5
- CSS3 (原生，无框架)
- JavaScript (ES6+)
- Chart.js (数据可视化)

## 快速开始

### 方式一：使用环境变量启动（推荐）

**配置环境变量：**
```bash
# 设置必要的环境变量
# SECRET_KEY: 应用密钥（生产环境必须设置复杂密钥）
# DATABASE_URL: 数据库连接URL（可选，默认使用SQLite）
# PORT: 应用端口号（可选，默认5000）

# Windows命令行示例
set SECRET_KEY=your_secure_secret_key_here
set PORT=5000

# Linux/Mac终端示例
export SECRET_KEY=your_secure_secret_key_here
export PORT=5000
```

**使用启动脚本：**

**Windows用户：**
```bash
# 在命令提示符或PowerShell中运行
start.bat
```

**Linux/Mac用户：**
```bash
chmod +x start.sh
./start.sh
```

### 方式二：手动启动

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 启动应用：
```bash
python app.py
```

3. 访问系统：
打开浏览器访问 `http://localhost:5000`

## 功能使用说明

### 1. 登录系统

1. 打开浏览器访问 `http://localhost:5000`
2. 输入用户名和密码
3. 点击"登录"按钮

### 2. 仪表板

登录后自动进入仪表板，可以看到：
- 总客户数、销售线索、销售机会、待办任务统计
- 销售漏斗图
- 关键指标（本月新增客户、销售机会总金额、线索转化率）
- 最近任务和最近客户列表

![Dashboard界面](screenshots/dashboard.png)

### 3. 客户管理

**添加客户：**
1. 点击左侧菜单"客户管理"
2. 点击右上角"添加客户"按钮
3. 填写客户信息（姓名为必填项）
4. 点击"保存"

**搜索客户：**
- 在搜索框输入关键词，自动搜索客户姓名、公司、邮箱

**筛选客户：**
- 使用状态下拉框筛选（活跃/不活跃/潜在）

**编辑客户：**
1. 点击客户行的"编辑"按钮
2. 修改信息
3. 点击"保存"

**删除客户：**
1. 点击客户行的"删除"按钮
2. 确认删除

### 4. 销售线索管理

**添加线索：**
1. 点击左侧菜单"销售线索"
2. 点击"添加线索"按钮
3. 填写线索信息
4. 设置线索评分（0-100）
5. 选择线索来源
6. 点击"保存"

**更新线索状态：**
- 新建 → 已联系 → 已确认 → 已转化/已失去

### 5. 销售机会管理

**看板视图：**
- 销售机会以看板形式展示
- 包含5个阶段：潜在客户、资格审查、提案、谈判、成交

**添加机会：**
1. 点击"添加机会"按钮
2. 选择关联客户
3. 填写机会名称、金额、成交概率
4. 设置预计成交日期
5. 点击"保存"

**拖拽更新阶段：**
- 直接拖动机会卡片到不同阶段列

### 6. 联系人管理

**添加联系人：**
1. 点击"添加联系人"按钮
2. 选择所属客户
3. 填写联系人信息
4. 可勾选"设为主要联系人"
5. 点击"保存"

### 7. 任务管理

**添加任务：**
1. 点击"添加任务"按钮
2. 填写任务标题和描述
3. 设置优先级（低/中/高/紧急）
4. 设置状态（待处理/进行中/已完成/已取消）
5. 设置截止日期
6. 点击"保存"

**筛选任务：**
- 按状态筛选
- 按优先级筛选

### 8. 主题切换

点击右下角的主题切换按钮（🌙/☀️）可以在深色和浅色主题之间切换。

## 常见问题

### Q: 如何注册新用户？
A: 点击登录页面的"立即注册"链接，填写用户名、邮箱和密码即可注册。

### Q: 忘记密码怎么办？
A: 当前版本暂不支持密码找回，可以使用默认管理员账户登录。

### Q: 数据存储在哪里？
A: 数据存储在SQLite数据库文件 `crm.db` 中，位于项目根目录。

### Q: 如何备份数据？
A: 复制 `crm.db` 文件即可备份所有数据。

### Q: 支持多用户吗？
A: 支持。每个用户只能看到自己创建的数据。

### Q: 如何导出数据？
A: 当前版本暂不支持数据导出，可以直接访问SQLite数据库文件。
